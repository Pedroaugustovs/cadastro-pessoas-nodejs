# Observação

A pasta node_modules não foi enviada para o repositório porque ela contém apenas as dependências do projeto. Essas dependências podem ser instaladas novamente utilizando o comando npm install.

Em alguns casos, no Windows pode aparecer um erro relacionado à execução de scripts no PowerShell. Caso isso aconteça, é possível resolver executando o comando abaixo no terminal:

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
